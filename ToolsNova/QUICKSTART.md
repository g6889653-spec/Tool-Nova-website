# ⚡ ToolNova - Quick Start Guide

Welcome to ToolNova! This is a collection of free online tools that run completely in your browser.

## 🚀 Get Started in 30 Seconds

### Option 1: Open Locally (Fastest Way)
1. Download all files from this repository
2. **Double-click `index.html`** - that's it! 🎉

### Option 2: Online (GitHub Pages)
Visit your deployed site at:
```
https://yourusername.github.io/toolnova
```
(Replace `yourusername` with your GitHub username)

### Option 3: Local Server (Recommended for Development)
```bash
# If you have Python installed
python -m http.server 8000

# Then visit: http://localhost:8000
```

## 📚 Available Tools

| Tool | Purpose | Key Features |
|------|---------|--------------|
| 🔐 Password Generator | Create strong passwords | Customize length, symbols, letters, numbers |
| 📝 Word Counter | Analyze text | Word count, reading time, statistics |
| 🔤 Text Case Converter | Convert text formats | camelCase, snake_case, UPPERCASE, etc. |

## 🔧 Want to Customize?

Edit these files:
- **Homepage**: `index.html`
- **Styles**: `css/style.css`
- **Logic**: `js/script.js` or tool files in `tools/`
- **Colors**: CSS variables at top of `style.css`

## 🌐 Deploy to The Web (5 minutes)

1. Create a GitHub account (free at github.com)
2. Create a new repository named `toolnova`
3. Upload all ToolNova files
4. Go to Settings → Pages → Enable GitHub Pages
5. Your site is live! 🎉

**Full instructions**: See `DEPLOYMENT.md`

- A: Yes! Download files and open `index.html` locally

**Q: Is my data secure?**
