📌 Quick Reference:
• lowercase: all letters reduced (good for URLs)
• UPPERCASE: all letters capitalized
• Title Case: first letter of each word capitalized
• camelCase: no spaces, first letter lowercase, then capitalize (for variables)
• PascalCase: like camelCase but first letter uppercase (for classes)
• snake_case: words separated by underscores (for databases)
• kebab-case: words separated by hyphens (for CSS & URLs)
